# 圖片上傳指南

如何使用腳本上傳圖片到 Cloudflare R2 CDN

---

## 快速開始

### 推薦方式：自動處理

1. **將圖片放在文章同目錄下**：
   ```
   content/2026/01/文章-slug/
   ├── 文章-slug.md
   ├── cover.jpg
   └── image1.png
   ```

2. **在文章中使用相對路徑**：
   ```markdown
   ---
   coverImage: "./cover.jpg"
   ---
   
   ![圖片](./image1.png)
   ```

3. **執行處理腳本**：
   ```bash
   npm run images:process
   ```

**腳本會自動完成：**
- ✅ 轉換為 WebP 格式
- ✅ 壓縮圖片
- ✅ 上傳到 R2 CDN
- ✅ 更新文章中的路徑
- ✅ 刪除本地圖片

---

## 配置環境變數

創建 `.env` 文件：

```env
# Cloudflare R2 配置
CF_ACCOUNT_ID=你的AccountID
R2_BUCKET=blog-post
R2_BASE_URL=https://img.waynspace.com

# R2 API 憑證
R2_ACCESS_KEY_ID=你的AccessKeyID
R2_SECRET_ACCESS_KEY=你的SecretAccessKey
```

**獲取 R2 API 憑證：**
1. 前往 [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 選擇 R2 > Manage R2 API Tokens
3. 創建新的 API Token
4. 複製 Access Key ID 和 Secret Access Key

---

## 圖片 URL 格式

處理後的圖片 URL：
```
https://img.waynspace.com/YYYY/MM/article-slug/image.webp
```

**範例：**
```
https://img.waynspace.com/2026/01/my-article/cover.webp
```

## 可用腳本

| 腳本 | 說明 |
|------|------|
| `npm run images:process` | 自動處理所有本地圖片（推薦） |
| `npm run images:verify` | 驗證 R2 上的圖片路徑 |
| `npm run images:delete-spaces` | 刪除 R2 上包含空格的文件 |

---

## 常見問題

**Q: 圖片顯示 404？**
- 確認已執行 `npm run images:process`
- 檢查 R2 bucket 是否啟用 Public Access
- 確認自定義域名 `img.waynspace.com` 已正確綁定

**Q: 圖片顯示 403？**
- 前往 Cloudflare Dashboard > R2 > Settings
- 啟用 "Public Access"

**Q: 圖片路徑包含空格？**
- 使用 `npm run images:process` 會自動清理空格
- 或使用 `npm run images:delete-spaces` 刪除 R2 上的問題文件

**Q: 如何手動上傳圖片？**
- 不推薦，建議使用自動處理腳本
- 如需手動，請參考 Cloudflare R2 官方文檔
